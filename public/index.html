Project: LX

LX is my custom language built on top of the LLVM toolchain. It is written in a combination of C++ and C#. Currently the language only supports basic functions and integer variable manipulation.

My end goal for the project is to be able to bootstrap the compiler - in other words write the compiler almost completly in LX. This project allows me to explore compiler design, language syntax design, code generation with LLVM Intermediate representation and cross-language interoperability.

Future goals for the project are to support more variable types (such as float or char), control flow structures, memory management and user-defined types. Some of the other features I would like to implement are a standard library, Visual Studio extension and a cutom linker written in Rust.

Project: MinOS

MinOS is my custom operating system written in ASM using the Fat12 file system. Currnetly it only supports printing strings to the console and reading data from the hard drive.

My end goal for the project is to be able to support C++11 allowing me to compile some apps on the operation system. This project allows me to explore computer architecthure, low level programming skills, memory management and kernel level development.

NOTE: this code is only designed to work on Linux (Debian).
